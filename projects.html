Food fire

A web application for food delivery using React, Tailwind CSS, and Redux. This project aimed to provide users with a seamless experience for browsing restaurants and placing orders online.

Technologies Used:

  • ReactJS: Frontend development

Screen Sync - Movie Recommendation Android Application

Screen Sync, an Android application designed for movie enthusiasts. It offers personalized movie recommendations based on user preferences and also showcases popular and upcoming movies. The app seamlessly integrates with a Flask API to fetch real-time movie recommendations, providing users with an engaging and tailored movie-watching experience.

Key Features:

  • Personalized movie recommendations using machine learning
  • Display of popular and upcoming movies
  • Integration with Flask API for real-time movie recommendations
  • User-friendly interface for intuitive navigation

Technologies Used:

  • Android Studio: Development environment for Android applications

  • Java: Programming language for Android app development

  • Flask: Python web framework for building APIs

  • Machine learning: Python libraries for movie recommendation

Arc Launcher - Android Launcher Application

Arc Launcher, an Android launcher application designed to customize and enhance the user's home screen experience. Arc Launcher allows users to personalize their device with custom widgets, app icons, and themes. Additionally, it integrates various APIs to provide real-time weather updates, calendar events, and news feeds directly on the home screen.

Key Features:

  • Customizable home screen with personalized widgets and themes
  • Integration with various APIs for weather updates, calendar events, and news feeds
  • User-friendly interface with intuitive navigation

Technologies Used:

  • Android Studio: Development environment for Android applications

  • Java: Programming language for Android app development

  • XML: Markup language for defining app layouts

  • API Integration: Utilized various APIs for weather, calendar, and news feeds
